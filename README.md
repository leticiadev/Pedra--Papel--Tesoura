✊✋✌️ Pedra, Papel e Tesoura ✊✋✌️

📌 Descrição do Projeto
Este projeto é um jogo simples e interativo de Pedra, Papel e Tesoura, desenvolvido com HTML, CSS e JavaScript.
A aplicação permite que o usuário escolha uma jogada clicando em um botão, enquanto o computador faz uma escolha aleatória. Em seguida, o sistema compara as jogadas e mostra na tela o resultado da partida.
O jogo possui uma interface estilizada, com botões interativos e exibição dinâmica do resultado.

🎯 Objetivo
O objetivo deste projeto é praticar conceitos fundamentais do desenvolvimento web, como:
Estruturação de páginas com HTML
Estilização com CSS
Uso de botões para interação com o usuário
Lógica de decisão com if / else
Geração de valores aleatórios com JavaScript
Manipulação do DOM para mostrar o resultado na tela

✅ Funcionalidades
📌 O sistema permite:
Escolher entre Pedra, Papel ou Tesoura
Sorteio automático da jogada do computador
Comparação das jogadas para definir o vencedor
Exibição do resultado na tela sem recarregar a página

🎮 Resultados possíveis:
🤝 Empate
🏆 Você ganhou
❌ Você perdeu

🛠 Tecnologias Utilizadas
HTML5
CSS3
JavaScript

⚙️ Como Funciona
Ao clicar em uma das opções, o sistema:
Recebe a jogada escolhida pelo usuário
Sorteia uma jogada aleatória para o computador
Compara as jogadas usando condições (if, else if, else)
Mostra o resultado final na tela com as escolhas e o vencedor
